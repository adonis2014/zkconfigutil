about

这是一个非常好用的使用zookeeper配置工程的组件，使用java注解实现，想对哪个类的哪个属性进行配置加上相应的注解即可，并且可以实现动态监听，假如你是分布式工程，在zookeeper上修改值会立即被通知到所有机器，在这个过程中您只需要实现需要配置的属性的resolve，并用zkconfigutil.register(class, true),实现注册即可

a simple case

` /**

  • just test

  • @author JERRY * */ @TypeZkConfigurable(path="/test/jerry", server="localhost:2181", useOwnZkServer=false) public final class Demo {

    @FieldZkConfigurable(resolve=DemoF1Resolve.class, dynamicUpdate=true) public static String F1 = "F1";

    @FieldZkConfigurable(resolve=DemoF2Resolve.class, dynamicUpdate=true) public static String F2 = "F2";

    @FieldZkConfigurable(resolve=DemoF3Resolve.class, dynamicUpdate=true) public static Boolean F3 = false; } `

ZkConfigUtil app = new ZkConfigUtil("localhost:2181"); app.register(Demo.class, true);
